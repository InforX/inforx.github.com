新型冠状病毒信息库 UCloud本地镜像

发表于 2020-02-03 | 分类于 学术资源 | 阅读数

本次公布的全量病毒、人类病例基因组数据是11GB左右，因为都在美国NCBI 库，访问下载只有10KB/S左右。 UCloud同事自发把这些数据全量都拉到ufile上。这样，国内的药研、科研要用直接下载，速度会快很多。

#说明

2019新型冠状病毒信息库（本库由中国国家生物信息中心（CNCB)/国家基因组科学数据中心(NGDC)建设），针对本次2019-nCoV新型冠状病毒，自2020年1月20日起，由国家生物信息中心维护并公开病毒基因序列库（https://bigd.big.ac.cn/ncov#）

其中，截止至2020年1月31日14时，共计累计基因序列67份，其中最重要的病毒基因核对样本MN908947，目前已于1月23日维护更新至第三版本（MN908947.3，获取地址：http://2019ncov-gene.ucloud365.com/2019-ncov/MN908947/MN908947.3.txt）

本病毒基因序列，可用于：

  • 通过高通量测序法，确定患者是否感染2019-nCoV新型冠状病毒，用于医学确诊。

  • 医学研究，用于研制临床药物、用于分析病毒株基因组成

  • 药物试验，疗程比对

  • 科学研究、生物药研制

  • 等等

目前所有病例序列均已镜像至UCloud UFile地址：http://2019ncov-gene.ucloud365.com/路径下，详见下表。

另包含6分完整人类RNA-seq序列（SRA序列），用于分析Coronavirinae、Bat SARS-like coronavirus、SARS coronavirus病毒与其他细胞、菌类、病毒在人体中的结合反应。

因SRA序列位于美国NCBI，且数据量大、传输时间久等不便性原因，目前已将截止至目前的2019-nCoV所有的SRA序列同步保持在UCloud Ufile公共服务中，可在国内无损访问。

# 使用方法

内网算力使用纪要测序数据【更快】

使用UCloud算力做生物科研计算时，可通过内网获取以上病毒基因序列、SRA基因序列。

使用内网域名前缀：http://2019ncov.ufile.cn-north-02.ucloud.cn/

例如：SRR10971381 【约2.6GB】

公网访问：http://2019ncov-gene.ucloud365.com//data/SRR10971381/SRR10971381【传输速度取决于互联网接入带宽】

内网访问：Windows Server或Linux均可

使用内网前缀http://2019ncov.ufile.cn-north-02.ucloud.cn/替换公网域名前缀http://2019ncov-gene.ucloud365.com/

文件名保持不变，仍为“/data/SRR10971381/SRR10971381”

构成内网访问完整域名：

http://2019ncov.ufile.cn-north-02.ucloud.cn//data/SRR10971381/SRR10971381

访问下载速度，可达50MB/S
