2019年前沿市场趋势洞察

发表于 2019-03-08 | 分类于 其他 | 阅读数

雷达传感技术

提起雷达技术，大众对此并不陌生。传统的雷达技术具有非常广泛的应用。但是，由于传统雷达设备的硬件成本高且体型巨大，因此一直在消费类电子产品中应用较少。近年来随着半导体技术的发展，雷达的尺寸、功耗和价格在不断下降。另一方面，毫米波、超宽带技术和多发多收的使用，结合创新的信号处理技术和计算能力不断增强的芯片，雷达的感知功能日益强大。

雷达感知具有许多独特的优势。例如，与视觉传感器相比，雷达不受光线明暗的影响，具有穿透遮挡物的能力，可以更好地保护个人隐私；与超声技术相比，雷达感知的距离更远，而且不会对人和动物造成伤害。

多年来，谷歌一直致力于这项技术的研究。谷歌的Project Soli是一项运用微型雷达监测空中手势动作的新型传感技术。这种特殊设计的雷达传感器可以追踪亚毫米精准度的高速运动，然后将雷达信号进行各种处理之后，识别成一系列通用的交互手势，方便控制各种可穿戴和微型设备。目前谷歌Project Soli手势传感器获美国FCC许可 。2017年，FCC悄悄地为“车载雷达系统”分配了专属频谱。Zendar正在利用这一点来制造雷达系统，为自动驾驶车辆提供恶劣天气下也能看到高分辨率画面的雷达技术，从而实现任何天气情况的自动驾驶。Anduril正在使用雷达传感技术来构建高科技高塔。它们可以识别正在移动的对象，如人和动物，覆盖范围最到可达2英里。值得一提的是，Anduril最大的支持者是创始人基金（Founders Fund），这是特朗普支持者彼得·蒂尔名下的风险投资公司。

集成软件帝国

Atrium正在通过技术打造法律界的特斯拉。它是专门针对律师事务所开发软件工具。这家企业的软件工具，有望借助机器学习技术来理解和分析各种法律文件，实现某些流程的自动化，随后再向外授权给律师事务所。Pilot正在为会计做同样的事情，而Triplebyte正在为招聘做这件事。

【任何公司都越来越需要发展其自身的软件业务。】

积木式编程

积木式编程让你忘记代码！像Zapier这样的面向企业的关联服务平台的公司允许非开放人员将不同的代码块捆绑在一起，比如编程组件加入乐高积木，组装乐高机器人等开发孩子无穷想象力。类似的如Airtable（airtable.com）和Notion（notion.so）允许组件式构建仅需要单击来存储和操作其数据。对于稍微复杂的用例，Retool（tryretool.com）允许任何人在其上构建交互式视图。由MIT媒体实验室和其终身幼儿园团队支持研发的Scratch (scratch.mit.edu) 算得上是中国地区最广为人知的。Scratch开创了积木式编程学习模式的先河。 微软公司的创始人比尔盖茨,在给儿童的编程课程上,就推荐使用积木式编程语言:Scratch。而这种积木式的学习模式的原型,是来自于Lego。现在，我们能够在不懂过多编程知识的情况下制作各种新奇的东西。简化软件变得非常重要。但目前对于积木式编程教育培训市场尚处于起步阶段，随着STEAM教育被写进国家政策、大综合考试开始试点编程，类乐高式的简易编程赛道逐渐热闹起来。

互联网在线学习社区

目前在网上出现了一代优秀创始人正试图创建永久的社区，这些社区是大牛的发源者，极大地促进了社会的发展。有一批新的网站试图将这种影响数字化，并在互联网上建立常青藤联盟校园。ProductHunt逐渐成为展示作品的默认场所，在国外很流行。IndieHackers一个为企业家提供知识共享的社区，Indie Hackers 通过采访有独立收入来源（收入来自客户，非风投注资）的企业创始人，记录下他们的创业经历，如何从零实现营收等故事，并免费提供给有需要的创业者。

分享经济带来的创业新机遇

通过使用软件和机器学习，在线房屋买卖平台Opendoor有效地建立一个技术驱动的业务模式，就像Uber和Airbnb一样，通过移动设备就能实现即时交易。 卡车界Uber-Convoy公司正在担任货运软件调度员，为卡车行业6000亿美元行业带来一场Uber式的变革。 Cloud Kitchens（由优步创始人特拉维斯卡兰尼克创办）简单来说，CloudKitchens被称为“WeWork的厨房版”，通常会接管闲置的房地产空间，为其配备厨房设施，并将它们出租给餐厅、食品和饮料公司。CloudKitchens则负责为他们提供房地产、设施管理、技术和营销等一系列服务。

【这一“云厨房”的概念正在特定地区获得成功，就比如印度。在那儿，传统餐厅模式对于餐厅主人的负担太重，因为租金、洗碗工、厨师再到服务员的工资都是一笔不小的开支。但通过使用“云厨房”的理念，这些餐馆老板可以与其他人共享厨房设施，并减少其他许多开支。】

人工智能芯片

全球GPU一直处于供应短缺的状态。GPU现在已成为人工智能的大脑，人工智能产业的发展带来了大量的需求。在AI 芯片大戏中，主角不仅是巨头，初创公司也都粉墨登场。像GraphCore，Groq，Rain，谷歌的TPU等公司正在竞相争取更多市场和更大利益。

开发者工具

Streamlit正在构建更好的Jupyter notebooks工具（数据科学/机器学习社区内一款非常流行的工具）。 Weights＆Biases为机器学习开发人员构建新工具，大幅简化机器学习软件开发流程，以便各行业能够解锁AI优势，并且不再局限于少数几家能够雇佣技术娴熟且非常昂贵的AI开发人员。 Rookout使您只需单击按钮即可调试代码。 Figma 一款基于浏览器的协作式 UI界面设计软件，与一般的 UI设计软件不同，它不仅拥有强大且智能的 UI设计功能，还完美支持多人协作功能，是第一款真正支持协作功能的 UI设计软件。
