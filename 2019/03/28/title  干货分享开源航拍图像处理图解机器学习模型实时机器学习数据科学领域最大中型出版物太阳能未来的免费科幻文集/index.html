OpenDroneMap一个开源的航拍图像处理工具

OpenDroneMap(ODM)是一个处理空中无人机图像的开源项目,可以把航拍图像进行点云、正射影像和高程模型等转换处理。 由OSGEO(开源地理空间基金会)的成员开发。
ODM是OSGEO开发的一个用于处理空中无人机图像的开源工具包的名称，这个工具包是在五年前开始的，去年取得了一些进展。它现在为无人机和其他低空图像收集(如气球和风筝)提供了现代摄影测量技术，并进行全自动匹配，数字表面建模和拼接。使用立体视觉或创建正射影像和地图，这是从场景的多个2D视图中提取3D信息的过程。

https://www.opendronemap.org

对于机器学习初学者来说理解人工智能领域的一些概念是相对困难的，国外大牛JayAlammar博客通过图解方式解释机器学习的概念，把模型拆开一步步带你读懂它是如何运作的。

http://jalammar.github.io

DARPA与美国国家科学基金会NSF合作开展“实时机器学习”（RTML）的新项目

据军事嵌入式网站mil-embedded报道，美国国防先期研究计划局（DARPA）正在和美国国家科学基金会（NSF）联手合作，开展一项名为“实时机器学习”（RTML）的新项目，该项目旨在通过开发一种基于机器学习（ML）框架自动生成新芯片的设计方法，针对新兴的ML应用量身定制专用集成电路（ASIC），从而降低与开发应用程序相关的成本。

亚利桑那州立大学推出了一本关于太阳能未来的免费科幻文集

亚利桑那州立大学最近发布了一本想象太阳能未来的文集。<>是关于“探索由太阳能驱动的人类未来”，并且是该大学科学与想象中心更广泛的出版计划的一部分，该计划自2011年以来发布了类似的关于太空乐观未来与气候变化的选集。《The Weight of Light》将提供免费下载，该中心网站提供了 ePub、 HTML、iBook和 Mobi版本。

https://csi.asu.edu/books/weight/

GitHub 上开源了一个名为 Handtrack.js 的项目，3 行 JS 代码搞定手部动作跟踪

近日，GitHub 上开源了一个名为 Handtrack.js 的项目，有了它，你只需要 3 行代码就能用来检测图片中手的动作。它可以让开发人员使用经过训练的手部检测模型快速创建手势交互原型。这个库的目标是隐藏与加载模型文件相关的步骤，为用户提供有用的函数，并允许用户在没有任何 ML 经验的情况下检测图像中的手，你不需要自己训练模型。你也无需导出任何图或已保存的模型。你可以直接在 Web 应用程序中包含 handtrack.js（详情如下），然后调用这个库提供的方法。

https://github.com/victordibia/handtrack.js/
