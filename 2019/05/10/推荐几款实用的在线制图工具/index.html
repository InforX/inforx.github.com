推荐几款实用的在线制图工具

发表于 2019-05-10 | 分类于 工具 | 阅读数

免费在线电路图设计工具

https://library.io/
免费在线电路图设计工具是一个基于网页的电路图仿真设计工具，用户只需推拽网站提供的电路元器件即可搭建成你所需要的电路图，简单省事。

免费在线制作海报风格工具

https://www.postermywall.com/
PosterMyWall:免费在线制作海报风格工具是一个国外免费的在线海报制作工具，它提供了免费的种类繁多的高品质海报背景，我们可以使用这些专业的设计把自己的许多照片，放在好看的背景中，并且还可以容易的抠图，添加各种特效。

数据可视化制作工具

http://www.axiis.org/
Axiis:数据可视化制作工具是一个数据可视化框架，于五月份使用MIT许可证发布。Axiis让开发人员通过简洁直观的标记，清晰明白地定义数据可视化方式。Axiis在设计上就非常强调代码优雅，可以让你的代码像输出的图形一样美观。

在线免费开源绘制图表工具

http://www.jscharts.com/
JsCharts:在线免费开源绘制图表工具是一个用来在浏览器上直接绘制图表的JavaScript工具包。支持柱状图，圆饼图以及简单的曲线图。直接将这个库插入网页，然后从 XML 文件或 JavaScript 数组变量调用数据。PNG 格式，兼容所有主要浏览器。

在线表格制作转图片工具

https://livegap.com/charts/
Charts Builder:在线表格制作转图片工具是一个可以在线制作电子表格样式并生成图片的工具，让你无需使用任何软件即可在线创作图表，然后生成图片并复制粘贴到各种文档或者通过电子邮件发送给其他用户查看。

在线简报图表制作平台

http://www.easel.ly/
Easelly 网站是一个提供免费云端服务的简报数据制作工具，如果你不懂设计、也不懂排版的话，可以通过该网站的三步骤就可以快速制作出来你想要的资讯图表展示文章。看图说话越来越收到大众和高层的青睐。

所见即所得绘图工具

https://www.omnigroup.com/omniGraffle/
OmniGraffle 是一个支持绘制图表、流程图、组织结构图、插图、头脑风暴图、思维导图、心智图、UI界面设计的多功能绘图工具，仅适合于Mac OS系统，具有采用拖放的所见即所得界面，操作灵活功能强大。

在线信息图表设计工具

https://infogram.com/
“inFogram”是一个支持在线制作响应式资讯图表设计的工具，用户可免费轻松的组合各种图表样式，这样的效果是可以根据显示器屏幕的大小自动调整适合的阅读效果，简单的拖拽功能即可完成。

在线流程图教学制作工具

https://www.iorad.com/
“iorad”是一个非常适合于学校教师、企业培训人员使用的教学流程图制作工具，使用者只需要自己完整的操作一遍，系统机会自动录下每一个步骤，自动截图、划线、标注说明，最后生成教学流程图。

MolView在线化学分子编辑器

http://molview.org/
「MolView」是一款免费开源的化学分子式编辑、查看、绘制工具，其中包含了化合物、蛋白质、光谱数据库，使用者可以轻松绘制想要的分子式，支持2D和3D效果图。
