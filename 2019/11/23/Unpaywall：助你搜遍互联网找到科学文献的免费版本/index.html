Unpaywall：助你搜遍互联网找到科学文献的免费版本

发表于 2019-11-23 | 分类于 工具 | 阅读数

科学文献出版是一个相当复杂的行业，这个行业当前存在的一个现象是，过去十年设置了收费壁垒的大型科学期刊正开始向更为开放的模式转变，尽管如此，你仍然很难在短时间内找到一篇科学文献的全文。Unpaywall [http://unpaywall.org ]是一个浏览器插件，可以识别你正在寻找的文献，然后确定它是否在互联网上的某个地方免费提供。

先将这个插件安装在 Firefox 或 Chrome 浏览器上，然后你在页面中对论文作出简要描述，或是显示文章的一部分，此时便会出现一个小锁的图标，告诉你是否能在别处免费找到这部分资源。

值得指出的是，Unpaywall 主要是一款名为 “oaDOI” [https://oadoi.org] 的工具的界面，oaDOI 可以收集并整理有关可公开获取的杂志和文章的数据库。为了确保同一时代的人可以看到自己的著作，科研人员往往会提交论文，以预先打印资源库（如 Arxiv），或是寄存于自己的页面上或是大学数据库中。

Unpaywall 和 oaDOI 工具都是由 Impactstory [https://impactstory.org/about ] 开发的，这是一个专注于解决科学文献开放获取问题的非盈利机构。当然，你可能要对科学文献最终版本的标题、格式、文件名等做一些修改。这两个工具可以让你做到这一切，不仅支持 DOI 数字，还能对文章标题和作者进行比对。

如果一篇论文有五个或六个作者，那么谁的页面或谁的实验室会保存这篇论文呢？这是耗时费力的一件事，而且往往搜索无果，我自己就不止一次请求科研人员直接给我拷贝资料。但是，满足爱管闲事的记者的要求并不是他们的工作！所以，Unpaywall 至少会成为一个对我来说至关重要的工具。

Unpaywall 在很大程度上酷似 RECAP [ https://free.law/recap ]，也是一种利用互联网资源改善知识获取方式的绝佳途径。RECAP 可以帮助你寻找合法的文献，然后上传至这些文献还不存在的地方。相比 Sci-Hub 上面不合法但极为实用的盗版论文，Unpaywall 在道德上不会遇到太大的麻烦。虽然并不是每一篇论文都会出现在别处，但我们的确可以在别的地方找到许多论文资源。

《自然》杂志指出，还有一些工具也在开发之中，它们使用 oaDOI 及其他开放获取数据库来实现不同的目的。正如我之前所说，大型科学期刊正开始向这种模式转变，而《自然》则是需要引领这场潮流的出版巨头之一。
