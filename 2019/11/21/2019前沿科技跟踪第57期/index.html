发表于 2019-11-21 | 分类于 科技前沿 | 阅读数

苹果更新“人人能编程”课程 并将于12月在Apple Store中提供编程课程

据外媒Macrumors报道，苹果公司周三宣布，其已经全面改革了“人人能编程”（Everyone Can Code ）课程，以将其推广给世界各地更多的中小学生。新课程包括更多的教师资源，新的学生指南以及更新的Swift Coding Club材料。更新后的资料建立在现有的互动难题和活动的基础上，使编码更易于接近，并与学生的日常生活联系在一起。苹果公司将“人人能编程拼图”添加到Swift Playgrounds的新学生指南中，并且每章都将使学生以核心编程概念为基础并尝试新技术。教师指南将为教师提供将编程带入课堂所需的工具。新课程包括“人人能创造”项目指南，苹果公司称该指南将帮助学生“通过绘画、音乐、视频和照片表达他们所学的知识”。根据苹果公司的说法，全球5000多家学校中的数百万学生在使用“人人能编程”课程。

日本拟用无人潜艇调查海底稀土 推进深海稀土泥开采技术开发

日本政府计划最早于2020年开始联合日本海洋研究开发机构等，利用自主式无人潜艇（AUV）调查位于小笠原诸岛南鸟岛的深海之中的稀土储量。据日经中文网20日报道，调查使用的自主式无人潜艇由美国伍兹霍尔海洋研究所开发。包括附属品在内，购买金额约为10亿日元（约合人民币6500万）。潜艇将在南鸟岛的日本专属经济区内的6000米深海向海底发射超声波，测量包含高浓度稀土的“稀土泥”的分布和厚度等。

卡内基梅隆大学开发VERA技术：利用手机视频确定枪手位置

据外媒报道，现在，当发生大规模枪击案时，越来越多的旁观者开始用智能手机录制视频。得益于一种新的系统–VERA，这些录像可以很快就能找到凶手的下落。据悉，这套系统由卡内基梅隆大学的一个团队开发，其需要三个或更多的智能手机同时在不同的地点记录下事件的视频/音频。首先，该系统能够根据已知建筑物和其他结构的记录图像来确定每个手机在环境中的位置。接下来，它会“监听”每台手机的音轨中的两种声音。第一种声音是当飞行的子弹打破音障时产生的超音速“裂纹”；另一种是枪口爆炸声，虽然这种声音实际上是在超音速裂纹发生之前发出，因为它只以声速传播所以它能瞬间到达手机麦克风。通过分析记录下来的裂纹和爆炸之间的时间，VERA能够确定手机离枪有多远。然后，通过比较三部手机的位置和枪支之间的距离，它也能确定枪声的具体方位。

科学家开发柔性机器人ElectroSkin：可模仿蜗牛爬墙壁

从毛毛虫和蜗牛上获得灵感，来自布里斯托大学的科学团队将这种特性融入到现代化机器人，制造了一种小型、有弹性的柔性机器人ElectroSkin。团队已经设想了ElectroSkin的诸多未来应用场景，其中包括部署在危险区域，甚至是作为自动墙壁清洁器。由于具有弹性，柔韧性，ElectroSkin可以卷起并塞入口袋中，在展开状态下可以扔到任意表面进行工作。在未来的开发中，科学团队设想如果装备了不同的传感器和附件，该机器人可以对墙壁、天花板进行清洁；爬进倒塌的建筑物内在人力所不能到达的地方传输视频信号。该小组的研究成果发表在《Soft Robotics.》杂志上。

三维晶体管阵列有望打破摩尔定律

目前，用于计算机处理器的硅集成电路正接近单个芯片上晶体管的最大可行密度，至少在二维阵列中是这样。摩尔定律看似已难以维持。美国密歇根大学一研究团队却另辟蹊径，将晶体管阵列带入三维空间，在最先进的硅芯片上直接堆叠第二层晶体管。这一研究为开发打破摩尔定律的硅集成电路铺平了道路。
