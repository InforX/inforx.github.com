腾讯全文检索引擎 wwsearch 正式开源

wwsearch为海量用户下的全文快速检索而设计，底层支持可插拔的lsm tree存储引擎，具备支持按用户的亿级分表、低延时、高效更新、索引压缩、功能丰富、内存消耗低等特点。目前覆盖企业微信所有在线检索场景， 最大业务场景有300亿+条记录，索引词项万亿+，存储容量几十TB。在大规模数据下，服务运行稳定，可以为业务增长提供稳定有力保障！现在，腾讯把wwsearch开源。希望wwsearch能被有相同业务场景需求的开发者使用，解决大家在全文检索场景遇到的类似问题。

Github 开源地址：https://github.com/Tencent/wwsearch

wwsearch 国内镜像地址https://git.code.tencent.com/Tencent_Open_Source/wwsearch

科学家正在研发一种可以从大炮中直接发射的变形无人机

据外媒The Verge报道，无人机是空中不可思议的有用机器，但是让它们起来并持续飞行可能很棘手，特别是在拥挤，多风或紧急情况下，而速度是一个重要因素。但是，来自加州理工大学和美国宇航局喷气推进实验室的一组研究人员提出了一种有趣的解决方案：从大炮中发射出无人机。工程师的作品叫做SQUID（精简快速展开调查无人机）。无人机全长不到一英尺（27厘米），重18盎司（530克），有四个弹簧加载的转子臂，在无人机发射后不到0.1秒就可以卡入到位。这不是有史以来第一架以弹道发射的无人机。但较早的示例，例如雷神公司的LOCUST装置，则使用固定翼而不是多旋翼设计，该设计具有更大的航程和稳定性，但操纵性却较差，而且很难飞行。

隐私导向搜索引擎Startpage宣布在搜索结果中加入“新闻”标签

Startpage是近年来随着爱德华·斯诺登揭露美国监控活动而流行起来的搜索引擎之一。Startpage通过不会储存用户IP地址、个人用户数据和跟踪cookie而在竞争中脱颖而出。另外，用户还可以Anonymous View（匿名查看）工具匿名查看结果。据外媒报道，日前，专注于用户隐私的搜索引擎Startpage宣布了在搜索结果中加入“新闻(News)”标签的消息。在此之前，Startpage只提供网页、图像和视频这三项标签，最新加入的新闻标签意味着该搜索引擎将跟其竞争对手更加接近。与其他搜索功能一样，新闻搜索结果也不会受到任何跟踪的影响，这确保了用户可以看到更平衡的结果列表。Startpage不会根据浏览历史来过滤新闻结果，而是根据发布的时间和日期来过滤，这样能够保证用户跟上外界最新的发展。

松下开发出可为移动中的搬运机器人无线充电技术

据日经中文网11月22日消息，松下开发出了一种能够给移动中的搬运机器人充电的无线供电技术。该技术需要在地面铺设薄金属板，当机器人从上方经过时进行充电，1秒钟可充电1千瓦。该技术利用了松下在家电开发中积累的经验，具有供电侧和受电侧即使出现位置上的偏差也能传输电力的特点，其将使物流仓库或工厂内的搬运机器人不再需要专门的充电场所和充电时间，将帮助无人仓库和工厂提高生产效率。

法国计划将氢气与天然气混合输送以减少碳排放

作为减少碳排放努力的一部分，法国天然气网络可能从2030年起进行调整，将天然气与20%氢气混合输送。法国天然气运营商表示，法国的天然气网络在初期可以输送含6%氢气的天然气混合物。他们建议政府到2030年将氢气含量设定为10%，再逐渐增加至20%。目前法国的氢气主要由石油和天然气生产，用于工业用途，其二氧化碳排放约占总排放量的3%左右。在其长期能源计划中，法国的目标是到2023年使氢气的生产脱碳率达10%，并在2019年拨出1亿欧元来鼓励氢气在社会生产领域，特别是在运输领域的使用。
