Netflix 开源内部 Python 工具 Metaflow：机器学习项目部署时间从四个月缩短至 7 天

近日，Netflix 数据科学团队宣布其内部使用多年的 Python 库 Metaflow 正式开源 [ https://github.com/Netflix/metaflow ]。Metaflow [ https://metaflow.org ] 是 Netflix 机器学习基础架构的关键部件，主要用于加速数据科学工作流的构建和部署，Netflix 希望通过开源 Metaflow 简化机器学习项目从原型阶段到生产阶段的过程，进而提高数据科学家的工作效率。

东南大学2019年最新研究生精品课程《知识图谱》资源分享

该课程系统地讲解了知识图谱相关的知识，涉及知识表示、知识建模、知识抽取基础（数据采、实体识别、关系抽取、事件抽取）、 知识融合、 知识图谱表示学习、知识存储、基于知识的智能问答、实体链接、知识推理等所有知识图谱相关的知识点。课程适用于研究生、对知识图谱感兴趣的研究人员和工程师。资源链接: http://1t.click/bvVU

一份《交互式线性代数》电子书免费开源，支持在线全交互式体验（含源码，离线也能看）

关于线性代数的重要性，很多做机器学习的同学可能会感同身受，一般线性代数教程大多是枯燥乏味的而用交互式可视化方法学习数学估计是很多同学梦寐以求的。之前也介绍过一本好玩的线性代数书籍。今天再给大家分享一份《交互式线性代数》教程，支持在线全交互式体验。这份线性代数教程有在线版。网址为：http://textbooks.math.gatech.edu/ila/index.html

斯坦福和加州大学洛杉矶分校都在用的线性代数教科书免费下载

这本斯坦福和UCLA都在用的线性代数教科书，书名为Introduction to Applied Linear Algebra —Vectors, Matrices, and Least Squares。这本书的最大的特征是面向应用，通过实际的例子来展示一些看似简单的方法，将抽象的线性代数知识转化为现实世界的应用。书中讨论的例子，也覆盖了各个领域，不仅有数据科学、机器学习和人工智能，还涉及信号和图像处理、导航和金融领域等等。美国数学协会 Reviews的评价称，这本书可以作为将线性代数应用到数据科学中的第一本教科书。

教科书下载：https://web.stanford.edu/~boyd/vmls/vmls.pdf

这两门课程也都有相应的主页，并提供了配套的PPT可供下载，还有相应的课堂作业。

斯坦福课程主页：https://web.stanford.edu/class/ee103/

UCLA课程主页：http://www.seas.ucla.edu/~vandenbe/ee133a.html

美国防部发布《2019财年年度财务报告》，重点关注太空和未来战争技术

美国防部近日发布《2019财年年度财务报告》。报告显示，2019财年，美国防部在太空和未来战争技术领域投资953亿美元。这些资金主要用于基础和应用技术研究，先进技术开发、原型制造，以及重大采办项目的设计和开发等。此外，这些资金还用于升级当前武器系统，以确保当前使用的武器系统和为未来开发的武器系统能够保持技术优势。
