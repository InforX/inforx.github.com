ArXiv Monitor跟踪ArXiv论文技术指标的搜索工具

ArXiv Monitor 【http://arxiv.aiindex.org】 是一个完整的论文搜索引擎工具，其目标是自动、持续地跟踪ArXiv上发表论文的技术指标。人工智能进度监测旨在提供跨任务、数据集、跨类别、技术指标以及其他相关类别的人工智能进度的高度概括。ArXiv Monitor由 HAI-AI Index开发，是一个开放的社区资源，为数据人工智能的技术讨论奠定基础。

Android 源代码搜索工具

一个可以在交叉的代码引用之间进行跳转的 Android 源代码搜索工具。众所周知 Android 的源代码的一系列 git 仓库是通过一个叫 repo 的工具来进行管理，查看源代码非常不方便，所以 AOSP 团队跟 Google 内部的 Code Search 团队合力一起开发了一个 Android 源代码搜索工具：https://cs.android.com

Facebook FastMRI 项目发布新的神经影像开源数据集

Facebook 人工智能实验室 FastMRI 项目项目组宣布了一个新的来自 NYU Langone Health 的开源数据集，基线模型以及新近扩展的研究论文，以帮助人工智能研究界加速并扩大该领域的研究。FastMRI 是 Facebook 人工智能实验室与纽约大学 Langone Health 医院展开的一项联合研究合作项目，旨在使用人工智能加速磁共振成像（MRI）扫描。据了解，该数据集包含了 6,970 个完全未识别的原始（k 空间）格式的 MRI 病例，以及 10,000 个包含 DICOM 格式的 370,000 个图像切片的 10,000 例。

GitHub 传送门：https://github.com/facebookresearch/fastMRI

Flutter: 首个面向环境计算打造的 UI 平台

在这个新兴世界中，大家关注的焦点开始从单一设备转向环境——一个允许您随时随地获取服务和软件的环境。我们将其称为环境计算 (ambient computing)，这同时也构成了 Flutter 愿景的核心: 一个用于构建精美体验的便携工具包，让开发者不论身处何处都可以在屏幕上绘制像素。在这个多设备、多平台的世界中，Flutter 旨在提供一套通用的框架与开发工具，让您免受设备与机型限制，顺利打造一流用户体验。由 Dart 驱动的 Flutter 引擎通过 stateful hot reload (保持应用状态的热重载) 功能帮助您实现快速开发，此外，无论是在移动端、桌面端，web 端还是嵌入式设备上，Flutter 引擎都能够提供出色的本地编译支持，从而为您带来快速强劲的性能表现。https://developers.googleblog.com/2019/12/flutter-ui-ambient-computing.html

下一代核电站将更小，更安全

人类对能源的需求日益增加，而旧的能源不仅已经面临开发殆尽的问题，还对环境和气候造成不好的影响。美国核能创业公司 NuScale Power 正在研发的小型模块化反应堆非常让人感兴趣，小身型、模块化、多用途，不仅可以用作发电，而且可以进行工业供热供汽，为城市供暖，还可用于海水淡化和海洋开发。迷你反应堆的含铀量要比传统反应堆少很多，这不仅降低了堆芯熔毁的风险，还限制了潜在事故的风险程度。这些更小、更安全的核反应堆可能使得小型反应堆能够进行大规模生产，可以降低核电行业的初始投资和财务风险。核能将轻易进入无碳能源的行列，对减缓气候变化的威胁还会有推动作用。在国际上，很多小型模块化反应堆已经正在筹备中了。据统计，全球共有约 50 种不同的小型模块化反应堆设计或概念处于发展和规划阶段。未来，这项技术将大大提高核电的可预测性、可靠性和规模经济效益，也为核电服务于小型市场创造了可能性。

世界上第一个水上浮动式奶牛场

目前，荷兰公司Beladon正在阿姆斯特丹附近建立世界上第一个水上浮动式奶牛场。据悉，这是一个多层次的高科技设施，将漂浮在鹿特丹的水面上。该农场长89英尺、宽89英尺，可容纳40头奶牛，平均每天可生产211加仑的牛奶。Beladon公司表示，它们建立这样一个水上浮动式农场的想法，来自飓风袭击这座城市时，使得成千上万的食品运输卡车停运，当地居民购买新鲜农产品变得非常困难。于是，该公司决定寻找一种新方式来帮助该城市生产靠近居民居住地的食物，以便更好地避免食品供应链中断。最终，该公司设计出了水上浮动式农场。该农场的底层用于机械加工和乳制品包装，二层是奶牛活动场所，以及机器人挤奶的工作间；第三层则是为奶牛种植草的地方。奶牛的大多数食物都来自城市废物，例如当地啤酒厂留下的谷物和工厂的副产品；而奶牛所产生的粪便将作为天然肥料出售。Beladon公司还标签，该公司将致力于开设更多的浮动式农场。除了生产乳制品外，该公司还在设计浮动式养鸡场和浮动垂直养殖温室。近期，可能会在新加坡和中国建立类似的农场。
