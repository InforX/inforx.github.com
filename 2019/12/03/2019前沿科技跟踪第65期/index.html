2019前沿科技跟踪第65期

发表于 2019-12-03 | 分类于 科技前沿 | 阅读数

世界最大望远镜SKA中国区域中心原型机研制成功

近日在上海进行的第六届全球SKA工程大会上，中科院上海天文台宣布其牵头推进的中国SKA区域中心原型机研制成功，这是世界首台SKA区域中心原型机。平方公里阵列射电望远镜（Square Kilometre Array，简称SKA）是国际天文学家们计划建造的世界最大综合孔径的射电望远镜，整个阵列延伸超过3000公里，综合口径达到1平方公里。

MIT研究人员发现或能缓解肾结石疼痛的“松弛”药物组合

肾结石是一种常见的健康问题，尽管其可以控制，但它们可能会使患者非常痛苦。现在，麻省理工学院的研究人员发现了两种药物的特殊组合，可以使尿道管松弛，使结石更容易通过。肾结石是在肾脏中形成的晶体物质。它们通常只是自行排出体外，但体验却远非令人愉悦-当它们通过输尿管移动时，会引起剧烈的疼痛。因此，对于这项新研究，科学家着手研究是否将肌肉松弛剂直接输送到输尿管可以使结石的疼痛减轻。为此，他们首先对实验室培养皿中生长的人输尿管细胞测试了18种不同的药物，并测量了每种药物对肌肉松弛的作用。经过广泛的搜索和分析，他们发现两种药物的效果特别好：硝苯地平（目前用于治疗高血压）和rho激酶抑制剂（用于治疗青光眼）。而这两种药物一起使用效果更好。

研究称每天只刷一次牙可能增加出现某些心脏问题的风险

据外媒报道，过去的研究将口腔卫生不良与患某些慢性健康状况的风险增加联系起来。但是一项新的研究警告说，每天只刷牙一次或两次可能无法充分保护人们的心脏。根据欧洲心脏病学会发布的一项新研究，人们可能需要每天至少刷牙3次，以显著降低发生某些心脏问题的风险。研究人员根据报告的口腔卫生习惯对参与者进行了评估，发现每天至少刷牙3次的人发生心房纤颤的风险降低了10％，发生心力衰竭的风险降低了12％。研究人员在考虑了其他可能的影响因素（例如年龄、体重和高血压等状况）后，发现这种降低的风险。研究人员推测，经常刷牙会减少在牙龈和牙齿之间发现的细菌数量，从而使它远离血液。过去的研究发现，口腔细菌可以进入血液，引发炎症。然后，炎症反应可能会导致其他长期健康状况，包括心脏病和痴呆症的发展。

亚马逊推出AWS量子计算服务Braket

继谷歌、微软、IBM 之后，亚马逊也于今日宣布了面向 AWS 用户的量子计算服务 Braket 。这项以量子状态的通用符号命名的服务，现已推出预览版本。不同的是，亚马逊没有自建量子计算机，而是与 D-Wave、IonQ 和 Rigetti 合作，并通过 AWS 云平台来提供相应系统的访问。此外，亚马逊还启动了 AWS 量子计算中心和量子解决方案实验室。借助 Braket，开发人员得以着手构建量子算法和基础应用程序，然后在 AWS 及其合作伙伴的量子硬件中展开仿真曾测试。https://aws.amazon.com/cn/quantum-solutions-lab/

科学家开发Liquid Health Check：只需一份血液样本就能检测多种健康因素

据外媒报道，现代医学的效率正在逐年提高，比如AI为医生在花更多时间跟病人交流以此来做出更准确的诊断铺平道路。现在，健康研究人员则开发出一种简化的血液检测方法，它可以让医生用一个血液样本就能检测出多种疾病和健康因素。在一项早期的概念验证研究中，来自SomaLogic HQ和加利福尼亚剑桥大学、旧金山大学的科学家们使用他们的新血液测试扫描了每个研究参与者的5000个蛋白质–参与人数大概有17000名。在通过使用统计分析和机器学习技术处理完这些数据之后，研究人员就能开发出针对各种潜在健康问题的“预测模型”。

科学家开发穿戴式Tip-Tap技术 可实现对计算机的无接触控制

据外媒New Atlas报道，考虑到外科医生在手术室中保持无菌状态的重要性，当他们进入手术室时，他们最好不要碰触脏的触摸屏。然而，一种新型的无线可穿戴设备可以使他们无需触摸屏幕即可浏览诸如术前计划图之类的数字内容。这种由加拿大滑铁卢大学开发的原型设备被称为Tip-Tap。它由定制的RFID（射频识别）标签组成，可轻松应用于一次性手术手套的表面，或直接以“电子纹身”的形式应用于皮肤。这种便宜标签的天线被分成两个单独的条带，其中一条位于拇指上，另一条沿着食指延伸。

快速再生原料制成环保高密纤维板

最近，德国斯图加特大学利用可快速再生的原材料，开发出一种新的无毒、完全柔性的纤维板（柔性HDF），可用于生产家具或室内装修。纤维板广泛用于家具行业或室内装修，根据制造工艺和密度的不同，通常要加入含有甲醛或异氰酸酯的树脂，而这两种物质均对人体有严重的健康风险。新型柔性纤维板另一优势是其硅酸盐含量可达干纤维重量的20%。由于硅酸盐是天然阻燃材料，因此添加纯矿物添加剂就可满足材料类别阻燃的德国标准。所使用的黏合剂是环境相容的热塑性弹性体。因此，所生产的纤维板实际上不含甲醛和异氰酸酯，从而在整个产品生命周期中将健康风险降至最低。以“Bioflexi”名称注册的这种环保材料价格低廉，这种新颖的高密度和柔性纤维板可用于生产各种形状的家具和隔板，以及具有防滑和抗冲击性能的地板。HDF柔性板已在美国、欧洲和马来西亚申请了专利。斯图加特大学委托技术许可机构进行该发明的产业化，并正在寻找合作伙伴以推向市场。

研究人员开发出混合抗生素：能穿过生物膜杀死细菌

据外媒报道，细菌本身并不难杀死。问题是，这些狡猾的细菌会在被称为生物膜的屏障后边建立起菌落，而专门用于对付它们的抗生素却很难穿透这些屏障。现在，来自昆士兰科技大学(QUT)的研究人员开发出了一种可以穿透一些生物膜的混合抗生素。据这项研究成员、研究论文首席作者Anthony Verderosa介绍称，生物膜是一种粘稠的涂层，通常情况下它可以阻止传统抗生素接触到细菌细胞，现在他们开发出的一种新型抗生素则可以骗过生物膜进而能够穿过它。据了解，这种新型抗生素是一种广泛使用的抗生素–环丙沙星和一种叫氮氧化物的化合物的混合物。后者最近被发现可以有效清除生物膜从而可以让环丙沙星进入并杀死细菌。研究人员表示，他们正在为下一步的人体试验做准备。相关研究报告已发表在《Antimicrobial Agents and Chemotherapy》上。
