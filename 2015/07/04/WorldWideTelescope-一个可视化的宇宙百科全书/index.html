WorldWideTelescope 一个可视化的宇宙百科全书

发表于 2015-07-04 | 分类于 开源项目 | 阅读数

我们在一个越来越开放的网络环境里，编程成为一种深入了解世界的方式。公开的数据和代码已经让编程这件事情成为站在巨人肩头的游戏。

在2008 年的一场加拿大蒙特利尔的 TED 大会上，微软发布了一个叫做 WorldWide Telescope 的项目。

通过使用哈勃天文望远镜和十多台地球卫星拍摄的画面，WorldWide Telescope 把太空中行星位置坐标的数据和它们的图像对应起来，通过在画面上添加图层，还能直接把用户连接到学术论文、维基百科、基本数据列表等深度信息上。WorldWide Telescope 成为了一个可视化的宇宙百科全书。

今年一月，微软首次公开了打算开源这个项目的打算，在 6 个月后，昨天，微软正式宣布，这个项目的代码已经逐渐放到了开源社区 Github 上，任何有兴趣的人都可以下载使用，在这个基础上做自己的改进。

过去WorldWide Telescope 一直提供 Windows 程序下载，也能在支持微软的 Sliverlight 平台的浏览器上打开。但很长时间以来都局限在 Windows 环境下。

为什么能够看得到的宇宙显得很重要？NASA 也一直在公开一部分空间探索数据，但是过去数据和图片并没有建立具体的空间联系。有这样一个成形的软件代码开源，在 Oculus Rift 消费者版就要上市的时候，也许很快我们就能有自己关于宇宙沉浸式的程序。

大公司开源已经成为一种常见的事情，微软在这一波公司里都算保守的。同一天，Google 开源了它的机器学习程序，能够把普通图片转换成使人产生幻觉的图像。目前 Facebook 已经在 Github 上有 200 多个开源项目，最新公布的是一个自动检查代码错误的软件。
